# Инструменты RAGFlow MCP

Сервер MCP в настоящее время предлагает специализированный инструмент для помощи пользователям в поиске релевантной информации, основанный на технологии RAGFlow DeepDoc (глубокий документ):

- **retrieve**: Извлекает релевантные фрагменты из указанных `dataset_ids` и необязательных `document_ids` с использованием интерфейса RAGFlow retrieve, на основе заданного вопроса. Подробная информация обо всех доступных наборах данных, а именно `id` и `description`, предоставляется в описании инструмента для каждого отдельного набора данных.

Для получения дополнительной информации смотрите нашу реализацию сервера [MCP](https://github.com/infiniflow/ragflow/blob/main/mcp/server/server.py) на Python.
