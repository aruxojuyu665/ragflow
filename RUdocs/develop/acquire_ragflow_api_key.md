# Получение API ключа RAGFlow

Для аутентификации ваших HTTP/Python или MCP запросов на сервере RAGFlow требуется API ключ. В этом документе приведены инструкции по получению API ключа RAGFlow.

1. Нажмите на ваш аватар в правом верхнем углу интерфейса RAGFlow, чтобы перейти на страницу конфигурации.
2. Нажмите **API**, чтобы перейти на страницу **API**.
3. Получите API ключ RAGFlow:

![ragflow_api_key](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/ragflow_api_key.jpg "API ключ RAGFlow")

:::tip NOTE
См. [RAGFlow HTTP API reference](../references/http_api_reference.md) или [RAGFlow Python API reference](../references/python_api_reference.md) для полного справочника по HTTP или Python API RAGFlow.
:::
