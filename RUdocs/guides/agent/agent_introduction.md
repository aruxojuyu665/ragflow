# Введение в агенты

Ключевые концепции, основные операции, краткий обзор редактора агентов.

## Ключевые концепции

Агенты и RAG (Retrieval-Augmented Generation — генерация с дополнением извлечением) — взаимодополняющие технологии, каждая из которых расширяет возможности другой в бизнес-приложениях. В версии RAGFlow v0.8.0 представлено агентное решение, включающее визуальный редактор рабочих процессов без кода на фронтенде и комплексный графовый фреймворк для оркестрации задач на бэкенде. Этот механизм построен поверх существующих RAG-решений RAGFlow и предназначен для оркестрации поисковых технологий, таких как классификация намерений запроса, ведение диалога и переписывание запросов, чтобы:

- Обеспечить более качественные результаты поиска и,
- Поддерживать более сложные сценарии.

## Создание агента

:::tip ПРИМЕЧАНИЕ

Перед началом убедитесь, что:  

1. Вы правильно настроили используемую LLM (Large Language Model — крупная языковая модель). Подробнее смотрите в руководствах [Настройка вашего API ключа](../models/llm_api_key_setup.md) или [Развертывание локальной LLM](../models/deploy_local_llm.mdx).
2. У вас настроен датасет и соответствующие файлы корректно распарсены. Подробнее смотрите в руководстве [Настройка датасета](../dataset/configure_knowledge_base.md).

:::

Нажмите вкладку **Agent** в верхней центральной части страницы, чтобы открыть страницу **Agent**. Как показано на скриншоте ниже, карточки на этой странице представляют созданных агентов, которых вы можете продолжить редактировать.

![Agent_list](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/agent_list.jpg "Список агентов")

Мы также предоставляем шаблоны, ориентированные на различные бизнес-сценарии. Вы можете либо создать агента на основе одного из наших шаблонов, либо создать его с нуля:

1. Нажмите **+ Create agent**, чтобы открыть страницу **agent template**:

   ![agent_template](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/agent_template_list.jpg "Список шаблонов агентов")

2. Чтобы создать агента с нуля, нажмите **Create Agent**. Либо, чтобы создать агента на основе одного из наших шаблонов, выберите нужную карточку, например **Deep Research**, задайте имя агента в появившемся диалоговом окне и нажмите **OK** для подтверждения.  

   *Вы будете перенаправлены на страницу **редактора рабочих процессов без кода**.*

   ![add_component](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/add_component.jpg "Добавление компонента")

3. Нажмите кнопку **+** на компоненте **Begin**, чтобы выбрать необходимые компоненты для вашего рабочего процесса.
4. Нажмите **Save** для применения изменений к вашему агенту.
