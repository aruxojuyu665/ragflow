# Компонент обработки текста

Компонент, который объединяет или разделяет тексты.

---

Компонент **Обработка текста (Text processing)** объединяет или разделяет тексты.

## Настройки

### Метод

- Split: Разделить текст
- Merge: Объединить текст

### Split_ref

Появляется только при выборе метода **Split**.

Переменная, которую необходимо разделить. Введите `/` для быстрого вставления переменных.

### Script

Шаблон для объединения. Появляется только при выборе метода **Merge**. Введите `/` для быстрого вставления переменных.

### Delimiters

Разделитель(и), используемые для разделения или объединения текста.

### Output

Глобальное имя переменной для вывода компонента, которая может быть использована другими компонентами в рабочем процессе.
