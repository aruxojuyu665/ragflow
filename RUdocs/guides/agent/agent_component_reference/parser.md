# Компонент Parser (Парсер)

Компонент, который задаёт правила парсинга (разбора) для вашего набора данных.

---

Компонент **Parser** определяет, как должны парситься различные типы файлов, включая методы парсинга для PDF, поля для парсинга в Email и методы OCR (оптического распознавания символов) для изображений.

## Сценарий

Компонент **Parser** автоматически добавляется на холст ingestion pipeline (конвейера загрузки) и требуется во всех рабочих процессах ingestion pipeline.
