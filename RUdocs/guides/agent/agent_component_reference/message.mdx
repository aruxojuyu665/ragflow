# Компонент Message (Сообщение)

Компонент, который отправляет статическое или динамическое сообщение.

---

В качестве заключительного компонента рабочего процесса, компонент Message возвращает окончательный выходной результат данных рабочего процесса вместе с предопределённым содержимым сообщения. Если предоставлено несколько сообщений, система выбирает одно из них случайным образом.

## Настройки

### Сообщения

Сообщение для отправки. Нажмите `(x)` или введите `/` для быстрого вставления переменных.

Нажмите **+ Add message** для добавления вариантов сообщений. Если предоставлено несколько сообщений, компонент **Message** случайным образом выбирает одно для отправки.
