# Компонент разбиения по заголовкам

Компонент, который разбивает текст на части по уровню заголовка.

---

Компонент **Token chunker** (разбиение по токенам) — это текстовый сплиттер, который использует указанный уровень заголовка в качестве разделителя для определения границ частей и создания чанков.

## Сценарий

Компонент **Title chunker** (разбиение по заголовкам) является необязательным и обычно размещается сразу после **Parser**.

:::caution WARNING
Размещение **Title chunker** после **Token chunker** является некорректным и приведет к ошибке. Обратите внимание, что это ограничение в настоящее время не контролируется системой и требует вашего внимания.
:::

## Настройки

### Иерархия

Определяет уровень заголовка для установки границ чанков:

- H1
- H2
- H3 (по умолчанию)
- H4

Нажмите **+ Add**, чтобы добавить уровни заголовков здесь или обновите соответствующие поля **Regular Expressions** для пользовательских шаблонов заголовков.

### Вывод

Глобальное имя переменной для вывода компонента **Title chunker**, которое может быть использовано последующими компонентами в pipeline загрузки данных.

- По умолчанию: `chunks`
- Тип: `Array<Object>`
