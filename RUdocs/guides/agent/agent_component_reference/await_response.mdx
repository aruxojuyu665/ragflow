# Компонент ожидания ответа

Компонент, который приостанавливает выполнение workflow (рабочего процесса) и ожидает ввод пользователя.

---

Компонент **Await response** приостанавливает workflow, инициируя диалог и собирая ключевую информацию через предопределённые формы.

## Сценарии

Компонент **Await response** необходим там, где требуется отображать ответы агента или организовать взаимодействие между пользователем и компьютером.

## Настройки

### Вопрос-подсказка

Определяет, показывать ли сообщение, заданное в поле **Message**.

### Сообщение

Статическое сообщение для отправки.

Нажмите **+ Add message**, чтобы добавить варианты сообщений. Если задано несколько сообщений, компонент **Message** случайным образом выбирает одно для отправки.

### Ввод

В компоненте **Await response** можно определить глобальные переменные, которые могут быть обязательными или необязательными. После настройки пользователь должен будет предоставить значения для этих переменных при взаимодействии с агентом. Нажмите **+**, чтобы добавить глобальную переменную с следующими атрибутами:

- **Name**: _Обязательно_  
  Описательное имя, дающее дополнительную информацию о переменной.  
- **Type**: _Обязательно_  
  Тип переменной:
  - **Single-line text**: Принимает одну строку текста без переносов.
  - **Paragraph text**: Принимает несколько строк текста, включая переносы.
  - **Dropdown options**: Требует от пользователя выбрать значение из выпадающего меню. Необходимо задать _как минимум_ один вариант для меню.
  - **file upload**: Требует загрузить один или несколько файлов.
  - **Number**: Принимает числовое значение.
  - **Boolean**: Требует переключения между включено и выключено.
- **Key**: _Обязательно_  
  Уникальное имя переменной.
- **Optional**: Переключатель, указывающий, является ли переменная необязательной.

:::tip NOTE
Для передачи параметров с клиента вызовите:

- HTTP метод [Converse with agent](../../../references/http_api_reference.md#converse-with-agent), или
- Python метод [Converse with agent](../../../references/python_api_reference.md#converse-with-agent).
  :::

:::danger IMPORTANT
Если вы установите тип ключа как **file**, убедитесь, что количество токенов в загруженном файле не превышает максимальный лимит токенов вашего провайдера модели; в противном случае plain text (простой текст) в вашем файле будет усечён и неполным.
:::
