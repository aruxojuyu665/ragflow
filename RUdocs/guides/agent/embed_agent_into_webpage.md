# Встраивание агента на веб-страницу

Вы можете использовать iframe для встраивания агента на стороннюю веб-страницу.

1. Перед началом необходимо [получить API ключ](../models/llm_api_key_setup.md); в противном случае появится сообщение об ошибке.
2. На странице **Agent** нажмите на нужного агента, чтобы перейти на страницу его редактирования.
3. Нажмите **Management > Embed into webpage** в правом верхнем углу холста, чтобы открыть окно с **iframe**:
4. Скопируйте iframe и встроите его на вашу веб-страницу.
