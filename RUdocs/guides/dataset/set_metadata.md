# Установка метаданных

Добавление метаданных к загруженному файлу

---

На странице **Dataset** вашего набора данных вы можете добавить метаданные к любому загруженному файлу. Этот подход позволяет «тегировать» дополнительную информацию, такую как URL, автор, дата и прочее, к существующему файлу. В AI-чатах с поддержкой LLM такая информация будет отправлена вместе с извлечёнными фрагментами для генерации контента.

Например, если у вас есть набор данных из HTML-файлов и вы хотите, чтобы LLM при ответе на ваш запрос указывал исходный URL, добавьте параметр `"url"` в метаданные каждого файла.

![Установка метаданных](https://raw.githubusercontent.com/infiniflow/ragflow-docs/main/images/set_metadata.jpg)

:::tip ПРИМЕЧАНИЕ
Убедитесь, что ваши метаданные имеют формат JSON; в противном случае изменения не будут применены.
:::

## Часто задаваемые вопросы

### Можно ли установить метаданные для нескольких документов одновременно?

Нет, метаданные необходимо устанавливать *индивидуально* для каждого документа, так как RAGFlow не поддерживает пакетное задание метаданных. Если вы считаете эту функцию важной, пожалуйста, [создайте issue](https://github.com/infiniflow/ragflow/issues), описав ваш кейс и его значимость.
