# Включение Excel2HTML

Преобразование сложных таблиц Excel в HTML-таблицы.

---

При использовании метода разбиения **General** можно включить переключатель **Excel to HTML** для преобразования файлов таблиц в HTML-таблицы. Если он отключен, таблицы из таблиц будут представлены в виде пар ключ-значение. Для сложных таблиц, которые нельзя просто представить таким образом, необходимо включить эту функцию.

:::caution WARNING
Функция отключена по умолчанию. Если ваш набор данных содержит таблицы с сложными таблицами и вы не включите эту функцию, RAGFlow не выдаст ошибку, но ваши таблицы, скорее всего, будут искажены.
:::

## Сценарии

Работает со сложными таблицами, которые нельзя представить в виде пар ключ-значение. Примеры включают таблицы с несколькими столбцами, таблицы с объединёнными ячейками или несколько таблиц на одном листе. В таких случаях рассмотрите возможность преобразования этих таблиц в HTML-таблицы.

## Особенности

- Функция Excel2HTML применяется только к файлам таблиц (XLSX или XLS (Excel 97-2003)).
- Эта функция связана с методом разбиения **General**. Другими словами, она доступна *только при* выборе метода разбиения **General**.
- При включении этой функции таблицы из таблиц с более чем 12 строками будут разбиваться на части по 12 строк каждая.

## Процедура

1. На странице **Configuration** вашего набора данных выберите метод разбиения **General**.

   _Появится переключатель **Excel to HTML**._

2. Включите **Excel to HTML**, если ваш набор данных содержит сложные таблицы, которые нельзя представить в виде пар ключ-значение.
3. Оставьте **Excel to HTML** отключённым, если в вашем наборе данных нет таблиц или если таблицы можно представить в виде пар ключ-значение.
4. Если ответы на вопросы по сложным таблицам неудовлетворительны, проверьте, включён ли **Excel to HTML**.

## Часто задаваемые вопросы

### Нужно ли включать эту функцию для PDF с сложными таблицами?

Нет. Эта функция применяется только к файлам таблиц. Включение **Excel to HTML** не влияет на ваши PDF.
