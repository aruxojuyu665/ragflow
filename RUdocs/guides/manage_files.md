# Файлы

Управление файлами в RAGFlow позволяет загружать файлы по одному или пакетно. Затем вы можете связать загруженный файл с несколькими целевыми наборами данных. В этом руководстве показаны базовые способы использования функции управления файлами.

:::info IMPORTANT
По сравнению с загрузкой файлов напрямую в различные наборы данных, загрузка их в управление файлами RAGFlow с последующей привязкой к разным наборам данных *не* является лишним шагом, особенно если вы хотите удалить некоторые распарсенные файлы или весь набор данных, но сохранить оригинальные файлы.
:::

## Создание папки

Управление файлами в RAGFlow позволяет создавать файловую систему с вложенной структурой папок. Чтобы создать папку в корневом каталоге RAGFlow:

![создать новую папку](https://github.com/infiniflow/ragflow/assets/93570324/3a37a5f4-43a6-426d-a62a-e5cd2ff7a533)

:::caution NOTE
Каждый набор данных в RAGFlow имеет соответствующую папку в директории **root/.knowledgebase**. Создавать подпапки внутри неё запрещено.
:::

![загрузка файла](https://github.com/infiniflow/ragflow/assets/93570324/5d7ded14-ce2b-4703-8567-9356a978f45c)

![пакетная загрузка](https://github.com/infiniflow/ragflow/assets/93570324/def0db55-824c-4236-b809-a98d8c8674e3)

## Предпросмотр файла

Управление файлами в RAGFlow поддерживает предпросмотр файлов следующих форматов:

- Документы (PDF, DOCS)
- Таблицы (XLSX)
- Изображения (JPEG, JPG, PNG, TIF, GIF)

![предпросмотр](https://github.com/infiniflow/ragflow/assets/93570324/2e931362-8bbf-482c-ac86-b68b09d331bc)

## Связывание файла с наборами данных

Управление файлами в RAGFlow позволяет *связывать* загруженный файл с несколькими наборами данных, создавая ссылку на файл в каждом целевом наборе данных. Таким образом, удаление файла в управлении файлами АВТОМАТИЧЕСКИ УДАЛИТ все связанные ссылки на файл во всех наборах данных.

![связать базу знаний](https://github.com/infiniflow/ragflow/assets/93570324/6c6b8db4-3269-4e35-9434-6089887e3e3f)

Вы можете связать файл с одним набором данных или с несколькими одновременно:

![связать несколько баз знаний](https://github.com/infiniflow/ragflow/assets/93570324/6c508803-fb1f-435d-b688-683066fd7fff)

## Удаление файлов или папок

Управление файлами в RAGFlow позволяет удалять файлы или папки по одному или пакетно.

Чтобы удалить файл или папку:

![удаление файла](https://github.com/infiniflow/ragflow/assets/93570324/85872728-125d-45e9-a0ee-21e9d4cedb8b)

Чтобы удалить файлы или папки пакетно:

![пакетное удаление](https://github.com/infiniflow/ragflow/assets/93570324/519b99ab-ec7f-4c8a-8cea-e0b6dcb3cb46)

> - Удалять папку **root/.knowledgebase** запрещено.
> - Удаление файлов, связанных с наборами данных, **АВТОМАТИЧЕСКИ УДАЛИТ** все соответствующие ссылки на файлы во всех наборах данных.
