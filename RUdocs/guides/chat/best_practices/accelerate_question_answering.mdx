# Ускорение ответа
import APITable from '@site/src/components/APITable';

Контрольный список для ускорения ответа на вопросы вашего чат-ассистента.

---

Обратите внимание, что некоторые ваши настройки могут занимать значительное время. Если вы часто сталкиваетесь с тем, что ответ на вопрос занимает много времени, вот контрольный список для рассмотрения:

- Отключение **Multi-turn optimization** (оптимизации многократного диалога) сократит время, необходимое для получения ответа от LLM.
- Оставление поля **Rerank model** (модель повторной сортировки) пустым значительно уменьшит время извлечения.
- Отключение переключателя **Reasoning** (рассуждения) сократит время размышления LLM. Для модели, такой как Qwen3, также необходимо добавить `/no_think` в системный prompt, чтобы отключить рассуждения.
- При использовании rerank модели убедитесь, что у вас есть GPU для ускорения; в противном случае процесс повторной сортировки будет *чрезвычайно* медленным.

:::tip NOTE 
Обратите внимание, что rerank модели необходимы в определённых сценариях. Всегда существует компромисс между скоростью и производительностью; вам нужно взвесить плюсы и минусы для вашего конкретного случая.
:::

- Отключение **Keyword analysis** (анализа ключевых слов) сократит время получения ответа от LLM.
- При общении с вашим чат-ассистентом нажмите на иконку лампочки над *текущим* диалогом и прокрутите всплывающее окно вниз, чтобы увидеть время, затраченное на каждую задачу:
